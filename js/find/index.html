JavaScript မှာ find() method သည် array ထဲက ပထမဆုံး element ကိုသာ ရှာဖွေပါတယ်။ ဒီ method ကို array ရဲ့ element တစ်ခုချင်းစီကို စစ်ပြီး condition ကို ဖြည့်စွက်တဲ့ ပထမဆုံး element ကို ထုတ်ယူနိုင်ပါတယ်။ မိမိလိုချင်တဲ့ value ရှာလျှင် " ပထမဆုံးတွေ့တဲ့ " element ကိုသာ ပြန်ပေးပြီး၊ မတွေ့ရင် undefined ပြန်ပေးပါတယ်။
