အားလုံး တူ ညီသောအခါမှာတော့ အားလုံးတစ်ခုတည်းသော condition ကို ဖြည့်စွက်ကြောင်းစစ်ဆေးရန် အသုံးပြုသည်။ every() method သည် array ထဲရှိ အားလုံးသော elements သည် တူညီသော condition ကို ဖြည့်စွက်ကြောင်းစစ်ဆေးရန် အသုံးပြုသည်။ ဥပမာအားဖြင့် တစ်ခုတည်းသော condition ကို array ထဲရှိ အားလုံး element များအပေါ်စစ်ဆေးလိုက်ပြီး အားလုံးပြည့်စုံပါက true ပြန်ပေးပြီး၊ တစ်ခု မပြည့်စုံပါက false ပြန်ပေးသည်။
